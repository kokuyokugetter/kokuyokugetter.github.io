健康

  • GronG(グロング) ホエイプロテイン100 スタンダード 抹茶風味 1kg amazon.co.jp
    • ココア味でもよかったが、もっと甘みを抑えたくなったので購入

ガジェット

  • iPad mini
    • iPad Proはあるけど、どこでも雑に使える読書用として追加購入。電子書籍（特に技術書）をローカルダウンロードしたり、漫画アプリなどを入れるつもり。漫画はスマホでも問題ないが、技術書をスマホで読むのは文字が小さくて辛かった。Proは重い（作業もできるようにSmart keyboard folioもついてるのでマジで重い）ので横になって読むのには相応しくないし、iPad miniで読みながらProでメモ取るのも可能になったのが良い。

PC周辺機器

  • Anker PowerWave+ Pad with Watch Holder（Apple Watch ホルダー付 5W / 7.5W / 10W ワイヤレス急速充電器）【Qi認証取得 / QC 3.0対応急速充電器付属】Apple Watch Series 4 / 3 / 2、 iPhone 11 / 11 Pro / 11 Pro Max / XS / XS Max / XR / X / 8 / 8 Plus　その他Qi対応機種各種対応 amazon.co.jp
    • 机の上にiPad、Macbook、Apple Watch、lightningのケーブルが散乱しているので、Apple Watchとlightningくらいはまとめてしまいたいということで購入。Apple Watchは当然として、lightningについてもiPhoneを充電するというよりはAirPodsの充電兼置き場という感じなのでワイヤレス充電でいいだろうという感じ
  • BUNKER RING 3 (全5色) バンカーリング iPhone/iPad/iPod/Galaxy/Xperia/スマートフォン・タブレットPCを指1本で保持・落下防止・スタンド(ジェットブラック) amazon.co.jp
    • 前述のiPad miniを1週間使っていて、手が小さいこともあって片手で画面を隠さずに持つのは難しいので、空中に保持するためのヘルパーとして購入。剥がして何度も貼れるそうなので、微調整が利きそうなのが嬉しい。その分接着強度は低いかもしれないが、常に手に持って振り回すようなスマホ用ではなく、読書用iPadなので問題ない。
  • Bauhutte(バウヒュッテ) 昇降式 L字デスク タワー (幅32×奥行67㎝) ブラック BHD-670T-BK amazon.co.jp
    • ゲーミング用品を取り扱う有名メーカーのラック。ゲーミングPCを置くというよりは、デスクの横にちょうどいいサイズの棚が欲しかったので購入。もちろんそのうちゲーミングPCを買うつもりはそれなりにあるので、最下段にはそれを置いたり、中段にモデムやルーターを置いたり、さらなる理想の環境構築を目指していく。組み立て時のドライバー回しが大変すぎた…マメもできた。
  • エレコム 電源タップ 一括&個別スイッチ 省エネ スイングプラグ 6個口 1m ホワイト T-E7A-2610WH amazon.co.jp
    • 昇降デスクに置く電源タップが3mだったけど、昇降するにしても1mあれば足りそうなことが判明したので、クソ邪魔な3mのものと交換する。
  • WORKOOL （2個セット） モニター台 モニタースタンド 机上台 電話台 プリンター台 デスク収納 キーボード収納 高さ調整可能 2個セット … amazon.co.jp
    • モニター台が欲しくて購入。後々モニターを2台にするつもりなので、2個セット。モニター台が欲しかった理由は、前にノートパソコン等があるとモニターの下半分が隠れるのと、モニターベタ置きだとスペースを食うため。

  • 回線工事(16000円程度？)
    • 部屋内で別のコンセントからインターネット回線を取ってこられるように工事を頼んだ。コレによってデスクの隣でSwitchで有線接続したり、ゲーミングPCを買った際も無理なく有線接続ができるようになった。前の位置だと3m程度のLANケーブルを無理やり引っ張る必要があって、かなり汚くなりそうだったので、仕方ない。元々前の位置にしたのは今の部屋配置にする前だったので、タイミングも良くなかった。
