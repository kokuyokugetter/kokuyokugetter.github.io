  • ゲーミングPC
    • メモリやストレージは多少積んで20万強。ゲームはするかもくらい。多用途に使います

  • エレコム USBハブ U3H-FC04BBK 【超小型・軽量設計】 USB3.0 Aポート×4 ケーブル1.5m ブラック MacBook/Surface/Chromebook他 ノートPC対応 amazon.co.jp
    • ゲーミングPCからデスク上にハブを伸ばすために購入したが、まだ使い道は特にない。しかし間違いなく使いそう
  • 【Amazon.co.jp限定】オーディオテクニカ プロフェッショナル モニターヘッドホン ATH-M20x/1.6 ケーブル長1.6m スタジオレコーディング / 楽器練習 / ミキシング / DJ / ゲーム / 有線 ATH-M20x/1.6 ブラック amazon.co.jp
    • ゲーミングPC用。Bluetooth欲しいけど付いてないから泣く泣く購入。あとたまにPCの振動拾うのか、キーンって音が鳴るのが悲しい。PCが悪いのかヘッドホンが悪いのか。
